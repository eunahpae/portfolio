노인 지원 웹 커뮤니티

개요 :
대한민국의 빠른 고령화로 인한 노인들의 사회적 고립 등의 문제를 해결하기 위해, 생활 정보 및 교육 지원을 받을 수 있도록 돕는 노인 사용자 친화적인 서비스 제작

주요역할 : 5명 중 팀장 (Full-Stack)

상세내용:
- Java, Spring, Oracle, HTML5, CSS3, JavaScript, STS3
- 관리자 페이지 : 회원현황 시각화, 회원관리, SMS&채팅 내역 / 수강신청 내역 확인
- SMS API 를 사용하여 문자발송 기능
- 3일 이상 미 출석시 안부문자 자동발송 기능 (사고방지 및 발견 목적)
- 웹소켓을 사용한 실시간 채팅 문의 기능
